# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

---

## 2️⃣ Backend Setup

```bash
cd Finguard
npm install
```

Create `.env`

```env
PORT=5000
JWT_SECRET=secretkey
MONGO_URI=YOUR_MONGODB_URI
```

Run backend:

```bash
npm start
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# 🐳 Docker Setup

Run backend container:

```bash
docker compose up
```
